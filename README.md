# Tech City Inventory Management System

## Project Overview
This is a C++ Inventory Management System designed for **Tech City**, an electronics retailer based in Vilnius, Lithuania. The software allows managers to add, delete, and view products across various categories including Smartphones, Laptops, Headphones, and Smartwatches.

## Technical Features
- **Polymorphism:** Uses a 3-tier inheritance hierarchy.
- **STL Integration:** Utilizes `std::vector` for storage and `std::sort` / `std::remove_if` for data management.
- **Exception Handling:** Custom error management for invalid product IDs and unsupported gadget types.
- **Generics:** Implemented a template-based `Inventory` class.
